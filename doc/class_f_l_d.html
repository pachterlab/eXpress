Detailed Description

fld.h express

Created by Adam Roberts on 3/20/11. Copyright 2011 Adam Roberts. All rights reserved. The FLD class keeps track of the observed fragment length distribution. It is initialized with a Gaussian prior with parameters specified by the arguments to the constructor. A small "Gaussian" kernel is then added for each observation. All mass values and probabilities are stored and returned in log space (except in to_string).
